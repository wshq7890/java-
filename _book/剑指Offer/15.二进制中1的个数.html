15.二进制中1的个数

题目描述

输入一个整数，输出该数二进制表示中 1 的个数。

可能死循环：n右移

如果n为负数，最高位表示符号，移位后最高位会设为1

常规解法："1"左移

初始让flag = 1，每次让n&flag ,如果结果不是0，说明有一个"1"

public int NumberOf1(int n) {
    int flag = 1;
    int count = 0;
    while(flag != 0){
        if((n & flag) != 0){
            count++;
        }
        flag = flag << 1;
    }
    return count;
}

推荐解法：n&(n-1)

该位运算去除 n 的位级表示中最低的那一位。

n       : 10110100
n-1     : 10110011
n&(n-1) : 10110000Copy to clipboardErrorCopied

时间复杂度：O(M)，其中 M 表示 1 的个数。

public int NumberOf1(int n) {
    int cnt = 0;
    while (n != 0) {
        cnt++;
        n &= (n - 1);
    }
    return cnt;
